Dinocyclopedia - A responsive, single-page dinosaur encyclopedia built with vanilla HTML, CSS, and JavaScript. Browse hundreds of dinosaur specimens with images, diet info, 
temporal ranges, and detailed descriptions — all fetched live from the [RESTasaurus API](https://restasaurus.onrender.com).

Note :  The website takes 15-30 seconds to load, please take some time to wait. Thank you :)
